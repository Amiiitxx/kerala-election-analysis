# 🗳 Kerala Lok Sabha Election Dashboard (Excel)

## 📌 Overview

This project is an interactive Excel dashboard analyzing the 2024 Kerala Lok Sabha elections, focusing on constituency-level insights and candidate comparison.

## 📊 Features

* Constituency analysis (Mavelikkara)
* Candidate comparison (UDF, LDF, NDA)
* Voter turnout insights
* Vote share visualization (2019 vs 2024)
* Clean and interactive dashboard design

## 📂 Files Included

* kerala-election-dashboard.xlsx
* dashboard.png

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Data Visualization
